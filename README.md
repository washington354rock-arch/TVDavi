# TVDavi - O seu Site de Notícias

Este é um site de notícias simples, criado em HTML, CSS e JavaScript puro.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Faça upload de toda a pasta `tvDavi`.
3. Vá em Configurações → Pages.
4. Selecione a branch main e a pasta root (/).
5. Salve e aguarde alguns minutos.
6. O site estará disponível em: https://SEU_USUARIO.github.io/NOME_DO_REPOSITORIO/
